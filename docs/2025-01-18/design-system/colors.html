Color Palette Guide

This guide explains how to apply a harmonious earth-tone color palette to the web application, maintaining consistency and readability across the interface.

üé® Palette Analysis

The palette contains warm, earthy tones that create an analogous and monochromatic harmony, ideal for elegant and sophisticated interfaces.

#3B0F0F

Dark Wine - Primary color for important actions and headings

#BDA09D

Light Rose - Soft backgrounds and subtle sections

#1A1A1A

Near Black - Main text color for maximum readability

#7E5B5A

Medium Rose Brown - Borders, hover states, secondary buttons

#A2A2A2

Neutral Gray - Placeholders, dividers, disabled states

#4D2626

Medium Wine - Primary buttons and focus states

#9C7876

Warm Intermediate - Subtle highlights and accents

Recommended color application by UI function:

Function Suggested Color CSS Example
Main page background #BDA09D background-color: #BDA09D;

Active checkbox or switch #4D2626 background-color: #4D2626;
Focus outline #9C7876 with transparency outline: rgba(156,120,118,0.5);

üåó Contrast Modes (Accessibility)

Important Accessibility Guidelines:
  • Avoid dark background + dark text (e.g., #3B0F0F on #7E5B5A)
  • For small text, maintain minimum contrast of 4.5:1 ‚Äî use #1A1A1A or white (#FFFFFF) depending on background
  • For large text (18pt+), minimum contrast of 3:1 is acceptable

Dark Mode Suggestion

For a dark mode variant, invert the color roles:

  • Background: #1A1A1A
  • Buttons: #7E5B5A
  • Text: #E9E5E5
  • Accents: #9C7876

‚ú® CSS Tokens / Tailwind Configuration

CSS Variables

:root { --color-bg: #BDA09D; --color-surface: #E9E5E5; --color-text: #1A1A1A; --color-text-secondary: #7E5B5A; --color-border: #A2A2A2; --color-primary: #4D2626; --color-primary-dark: #3B0F0F; --color-accent: #9C7876; }

Tailwind Configuration

theme: { extend: { colors: { primary: '#4D2626', 'primary-dark': '#3B0F0F', accent: '#9C7876', surface: '#E9E5E5', text: '#1A1A1A', 'text-secondary': '#7E5B5A', border: '#A2A2A2', background: '#BDA09D', }, }, }

Usage Examples
