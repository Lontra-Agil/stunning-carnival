üîÑ Detailed User Flows

Public Pages Flow

Landing / Home Page

Entry point for all users. Showcases platform features and value proposition.

Public Blog

Content marketing, tutorials, and platform updates accessible to everyone.

Terms of Service & Privacy Policy

Legal documentation and privacy information for transparency and compliance.

Authentication Flow

Sign Up / Login

Unified authentication entry point supporting multiple OAuth providers.

OAuth Providers

Google OAuth: Sign in with Google account
Apple OAuth: Sign in with Apple ID

Role Detection

System determines user role (Contractor or Developer) and routes to appropriate dashboard.

Contractor Flow (Hiring Path)

1. Talent Marketplace

Browse available developers and organizations with advanced filtering options.

2. Profile & Metrics

View detailed developer profiles including repository analytics and performance metrics.

3. Repository Analytics

Code Quality: Code review scores, best practices adherence
Performance: Commit frequency, merge success rate
Activity: Contribution trends, project engagement

4. Initiate Hire

Choose between individual developer contracts or team/organization contracts.

5. Payment Processing

Secure Stripe checkout integration for contract payments.

Developer Flow (Profile Management Path)

1. Developer Dashboard

Central hub for managing profile, repositories, and visibility settings.

2. GitHub Integration

Connect GitHub account to automatically import repositories and organizations.

3. Unified Repository List

Fetch all repositories from personal account and associated organizations in one view.

4. Visibility Control

Toggle Visibility: Choose which repos appear on public profile
Auto-discovery: System automatically links related projects
Metrics: Combined analytics across all visible repositories

5. Public Profile

Updated profile visible to contractors in the talent marketplace.

üîë Key Features

Platform Highlights:
  • Seamless Authentication: OAuth integration with major providers
  • GitHub Deep Integration: Automatic repository analysis and metrics calculation
  • Flexible Visibility: Developers control what information is publicly visible
  • Comprehensive Analytics: Code quality, performance, and activity metrics
  • Dual Contract Types: Support for individual and team/organization hiring
  • Secure Payments: Industry-standard Stripe integration
